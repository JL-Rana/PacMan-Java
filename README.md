# PacMan Java Game

A simple Pac-Man game developed using Java Swing.

## Features

* Pac-Man movement using keyboard controls
* Ghost movement with random directions
* Wall collision detection
* Food collection system
* Score and lives system
* Game over and restart functionality
* Object-oriented project structure

---

## Technologies Used

* Java
* Java Swing
* OOP Concepts
* IntelliJ IDEA

---

## OOP Concepts Used

* Encapsulation
* Abstraction
* Inheritance
* Polymorphism
* Composition
* Event-Driven Programming

## How to Run

1. Clone the repository

```bash
git clone https://github.com/JL-Rana/PacMan-Java.git
```

2. Open the project in IntelliJ IDEA

3. Run `Main.java`

## Learning Experience

I developed this project with the help of a YouTube tutorial and a GitHub repository (the repository link is included below). In the original tutorial, the entire project was written in a single file. However, I tried to organize the project by separating different logics into different files and classes. While doing this, I faced several challenges related to accessing objects, methods, and variables between files, as well as understanding constructors, polymorphism, and access modifiers.

This project helped me improve my understanding of Java Swing, object-oriented programming concepts, event handling, collision detection, and project organization.
